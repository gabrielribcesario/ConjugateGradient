# ConjugateGradient
A C implementation of the conjugate gradient method.

It is meant as a simple alternative to direct solvers in 
some languages such as Python and LabVIEW.

It uses AVX512 instructions for faster execution.

I plan on adding a preconditioned version of it 
to this repository. I also plan on creating other 
repositories composed of C implementations
of other krylov subspace solvers.
